\bx{
If we have that $(x_n) \rightarrow x$, then we can make $\abs{x_n - x}$ arbitrarily small. Consider
\begin{align*}
	\abs{x_n - x_m} = \abs{x_n - x + x - x_m} \\
	&\leq \abs{x_n - x} + \abs{x_m - x} \tag{Triangle Inequality} \\
	&< \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon
\end{align*}
}
